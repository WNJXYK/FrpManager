# FRP Manager

一个用于管理 FRP 客户端的 MacOS 应用程序。

## 功能特性

- 系统托盘常驻
- 管理多个 FRP 客户端配置
- 启动/停止 FRP 客户端
- 编辑 FRP 配置文件
- 查看 FRP 运行日志

## 开发

### 环境要求

- Node.js >= 14
- npm >= 6

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

### 构建应用

```bash
npm run package
```

## 使用说明

1. 点击系统托盘图标可以快速启动/停止 FRP 客户端
2. 点击"主界面"可以打开配置管理界面
3. 在配置管理界面可以：
   - 添加新的 FRP 配置
   - 编辑现有配置
   - 删除配置
   - 查看运行日志
